wifi.json easy setup

This page allows you to enter your WiFi network's SSID and password to generate the wifi.json configuration file. This file is essential for your P1Touch Screen to connect to your WiFi network during installation.

* Note: Modifying the WiFi Timeout is recommended only if you encounter Wi-Fi connection problems. Leave as is if you don't experience connectivity issues.

* Note: Adjusting the "colboot" value is recommended specifically in situations where you encounter issues during the printer power-on process, leading to a reboot of the p1touch screen because the printer is not ready. The "colboot" value allows you to define the number of seconds the system should wait for initialization to complete after a cold boot.
